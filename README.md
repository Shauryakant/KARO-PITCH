# Karo Pitch - Startup Ecosystem Landing Page

## About This Project

This landing page was designed and built to serve as a modern, investor-focused platform where early-stage founders from across India—especially Tier-2 and Tier-3 cities—can pitch their startups directly to investors. 

### AI Tools Used
This project was entirely generated and refined using **Antigravity**, an advanced agentic AI coding assistant developed by Google DeepMind. The AI handled the planning, UI/UX design, layout structuring, CSS styling, responsive mobile adaptations, and JavaScript interactions. Additionally, an integrated AI image generation tool was used to create the photorealistic, professional placeholder images of startup founders and metrics dashboards seen throughout the page.

### Design Rationale
The design approach was intentionally crafted to achieve several specific goals:

*   **Premium Ecosystem Vibe:** The aesthetic was modeled after top-tier startup platforms like Product Hunt, YC Demo Day, and Web3 launch pages. We wanted it to feel simultaneously professional for investors and inspiring for ambitious founders.
*   **Color Palette:** We opted for a "dark mode" deep navy background with vibrant, glowing accent colors (electric blue, purple, and orange). This creates a highly modern, sleek look that makes the content and startup cards pop, conveying a sense of cutting-edge innovation.
*   **Visual Storytelling & Interactivity:** To keep users engaged, we implemented smooth scroll reveals (`IntersectionObserver`), animated metric counters, and floating CSS keyframe animations on the hero cards. These micro-interactions build credibility and make the platform feel dynamic and "alive."
*   **Mobile-First Accessibility:** Knowing that many founders in Tier-2/3 cities primarily access the web via smartphones, the entire layout was built mobile-first. CSS Grid and Flexbox were heavily utilized to ensure the site looks perfect and remains easy to navigate on any screen size.
*   **Discovery Focused:** We evolved the standard "Featured Startups" section into an interactive "Discovery" grid with live category filters (SaaS, D2C, FinTech, etc.), allowing investors to actively explore the ecosystem without leaving the landing page.
